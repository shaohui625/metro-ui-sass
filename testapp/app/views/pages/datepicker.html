DatePickercomponent

THis component required metro-calendar.js and metro-datepicker.js 

  1. <div class="input-control text">
  2.     <input type="text">
  3.     <button class="btn-date"></button>
  4. </div>

Init with data-* API: 

  1. <div class="input-control text" data-role="datepicker">
  2.     <input type="text">
  3.     <button class="btn-date"></button>
  4. </div>

Init with Javascript: 

  1. <div class="input-control text" id="datepicker">
  2.     <input type="text">
  3.     <button class="btn-date"></button>
  4. </div>
  5.  
  6. <script>
  7.     $("#datepicker").datepicker();
  8. </script>

To set params you can use data-* api or javascript:

  1. // Data-* API
  2. <div class="input-control text" data-role="datepicker"
  3.     data-date="2013-01-01"
  4.     data-format="d/m/y"
  5.     data-position="top|bottom"
  6.     data-effect="none|slide|fade">
  7.     <input type="text">
  8.     <button class="btn-date"></button>
  9. </div>
  10.  
  11. // Javascript
  12. <script>
  13.     $("#datepicker").datepicker({
  14.         date: "2013-01-01", // set init date
  15.         format: "d/m/y", // set output format
  16.         effect: "none", // none, slide, fade
  17.         position: "bottom", // top or bottom
  18.     });
  19. </script>
