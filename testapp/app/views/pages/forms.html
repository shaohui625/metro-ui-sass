Input validation states

You can set validation state for input with build-in classes as additional class for .input-control: .warning-state, .error-state, .info-state, .success-state.
Also, you can set data-state attribute for input, select or textarea with next values: warning, error, info and success. 

  1. <div class="input-control text warning-state">
  2.     <input type="text" />
  3. </div>
  4.  
  5. <div class="input-control text">
  6.     <input type="text" data-state="success"/>
  7. </div>

Input control plugin

Input control plugin metro-input-control.js provides functionality for clearing inputs, show password and input file.

Input Auto Transform

Now input control plugin contains widget for form input auto transformation. 

  1. from:
  2. <input type="file" data-transform="input-control" />
  3.  
  4. to:
  5. <div class="input-control file">
  6.     <input type="file" />
  7.     <button class="btn-file"></button>
  8. </div>

Now supported:

  1. <input type="text" data-transform="input-control" />
  2. <input type="password" data-transform="input-control" />
  3. <input type="file" data-transform="input-control" />
  4. <input type="email" data-transform="input-control" />
  5. <input type="tel" data-transform="input-control" />
  6. <input type="checkbox" data-transform="input-control" />
  7. <input type="checkbox" data-transform="input-control" data-transform-type="switch" />
  8. <input type="radio" data-transform="input-control" />
  9. <select data-transform="input-control" />
  10. <textarea data-transform="input-control" />
