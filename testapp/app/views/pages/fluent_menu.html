Menu elements

Special elements:

  • fluent-button
  • fluent-big-button
  • fluent-tool-button

To create buttons segment in group you can use div with class segment. To create iconic button you can use span with class icon-* or img.

  1. <div class="tab-group-content">
  2.     <button class="fluent-big-button"><span class="icon-mail"></span>Create<br />message</button>
  3.     <div class="tab-content-segment">
  4.         <button class="fluent-big-button dropdown-toggle">
  5.             <span class="icon-pictures"></span>
  6.             <span class="button-label">Create<br />element</span>
  7.         </button>
  8.         <ul class="dropdown-menu" data-role="dropdown">
  9.             <li><a href="#">Message</a></li>
  10.             <li><a href="#">Event</a></li>
  11.             <li><a href="#">Meeting</a></li>
  12.             <li><a href="#">Contact</a></li>
  13.         </ul>
  14.     </div>
  15.     <div class="tab-content-segment">
  16.         <button class="fluent-big-button">
  17.             <span class="icon-cancel"></span>
  18.             <span class="button-label">Delete</span>
  19.         </button>
  20.     </div>
  21.     <div class="tab-content-segment">
  22.         <button class="fluent-tool-button"><img src="http://metroui.org.ua/images/Notebook-Save.png"></button>
