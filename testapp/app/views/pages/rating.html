Ratingcomponent

Rating is a component that generates a customizable star rating. This component required metro-rating.js.

  4.         <li class="rated"></li>

  6.         <li></li>
  7.         <li></li>
  8.     </ul>
  9.     <span class="score-hint"></span>
  10. </div>

Component definition

  1. <div class="rating"></div>

You can activate widget manually or auto with data-* api.

Activate with API on DATA-ROLE attribute. 

  1. <div class="rating"
  2.     data-role="rating"
  3.     data-static="false"
  4.     data-score="3"
  5.     data-stars="5"
  6.     data-show-score="true"
  7.     data-score-hint="Value: ">
  8. </div>

Activate with javascript. 

  1. $(function(){
  2.     $("#rating_1").rating({
  3.         static: false,
  4.         score: 2,
  5.         stars: 5,
  6.         showHint: true,
  7.         hints: ['bad', 'poor', 'regular', 'good', 'gorgeous'],
  8.         showScore: true,
  9.         scoreHint: "Current score: ",
  10.     });
  11. });

Change Rating color

To change color of rating you can use built-in color classes fg-* (see colors on Global styles page). Also you can change color style for rating element. 

  1. <div class="rating fg-green"></div>
  2. <div class="rating" style="color: #2784A7"></div>

Change Rating size

To change size of rating you can use built-in classes large or small. 

  1. <div class="rating"></div>
  2. <div class="rating large"></div>
  3. <div class="rating small"></div>
