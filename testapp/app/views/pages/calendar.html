Calendarcomponent

This component required metro-calendar.js. 

  1. <div class="calendar"></div>
  2. <div class="calendar" data-start-mode="month"></div>
  3. <div class="calendar" data-start-mode="year"></div>

Using

You can activate calendar manually (with javascript) or auto with data-role.

Auto

  1. <div class="calendar" data-role="calendar"></div>

Manually

  1. $("component_id").calendar();

Manually with options

  1. $("component_id").calendar({
  2.     format: string, //default 'yyyy-mm-dd'
  3.     multiSelect: boolean, //default true (multi select date)
  4.     startMode: 'day', //year, month, day
  5.     date: string, //the init calendar date (example: '2013-01-01' or '2012-01')
  6.     months: array, // array of month names from Jan to Dec
  7.     weekDays: array, // array of week day names from Sun to Sat
  8.     getDates: function(d){...}, // see example below
  9.     click: function(d){...}, // fired when user clicked on day, in "d" stored date
  10. });
