Images

General

Add classes to an <img> element to easily style images in any project.

  1. <img src="..." class="rounded">
  2. <img src="..." class="cycle">
  3. <img src="..." class="polaroid">
  4. <img src="..." class="rounded polaroid">
  5. <img src="..." class="cycle polaroid">
  6. <img src="..." class="shadow">

Border color

You can change border color with build-in classes bd-*

  1. <img src="..." class="rounded bd-amber">
  2. <img src="..." class="polaroid bd-white shadow">
  3. <img src="..." class="polaroid bd-red bg-pink">

Background color

You can change background color for polaroid style with build-in classes bg-*

  1. <img src="..." class="rounded bg-cobalt">
  2. <img src="..." class="polaroid bg-amber bd-white shadow">
  3. <img src="..." class="polaroid bg-lightBlue bd-white shadow">

Image size

Also you can set width of image with <span1>...<span12> classes.

  1. <img src="..." class="span2">
  2. <img src="..." class="span3">
  3. <img src="..." class="span4">

Image container

You can use class .image-container to create additional image presentation.
