Progress Barcomponent

This component require metro-progressbar.js. 

  1. <div class="progress-bar" data-role="progress-bar" data-value="10"></div>
  2. <div class="progress-bar" data-role="progress-bar" data-value="20" data-color="bg-green"></div>
  3. <div class="progress-bar" data-role="progress-bar" data-value="50" data-color="bg-red"></div>
  4. <div class="progress-bar" data-role="progress-bar" data-value="75" data-color="bg-pink"></div>
  5. <div class="progress-bar" data-role="progress-bar" data-value="100" data-color="#ccc"></div>

Any size for progress bar

  1. <div class="progress-bar small" data-role="progress-bar" data-value="10"></div>
  2. <div class="progress-bar" data-role="progress-bar" data-value="20" data-color="bg-green"></div>
  3. <div class="progress-bar large" data-role="progress-bar" data-value="50"></div>

You can activate widget manually or auto with data-* api.

Activate widget with Javascript

  1. $("#componennt_id").progressbar({
  2.     value: int // progress value, default 0
  3.     color: string, // existing class (bg-*) or color schema (#NNNNNN)
  4.     onchange: function(val){...}
  5. });

Set value with Javascript

  1. var pb = $("#componennt_id").progressbar();
  2.     pb.progressbar('value', 75);

Set color with Javascript

  1. var pb = $("#componennt_id").progressbar();
  2.     pb.progressbar('color', '#000');

Activate widget with API

  1. <div class="progress-bar" data-role="progress-bar" data-value="75" data-color="bg-pink"></div>
